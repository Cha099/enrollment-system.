SOFTWARE SPECIFICATIONS


The software should be easy to execute and the flow of its program are easy to understand, also its performance should be good in order to execute the proposed system. Below are the different software specifications that the system will need to be fullyoperational:


Frontend or Client–side


As shown in Figure 29 the researchers uses Bootstrap, JavaScript, CSS, HTML, and JQuery for frontend or client – side when implementing the proposed system. Bootstrap is a frontend framework for faster and easier for web development. Bootstrapalso includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels and many other, as well as optional JavaScript plug-ins. HTML – stands for Hypertext Markup Language and it allows theuser to create and structure sections, paragraph, headings, links, and blockhouses for webpages and applications. CSS on the other hand is the language for describingthepresentation of web pages, including colors, layout, and fonts. It allows one to adapt thepresentation to different types of devices, such as large screen, small screens, or printers. JavaScript is a text-based programming language used both on the client-side and server-side that allows making web pages interactive by the use of JQuery it is mucheasier tomake JavaScript. Where HTML and CSS are languages that give structure andstyle to web-pages, JavaScript gives web pages interactive elements that engage a user.


Backend or Server–side


The researchers also use C#.NET, MSSQL for database, and Internet InformationServices (IIS)for back – end / server- side in the said proposed system. .NETis anapplication development framework – it contains numerous libraries containing a rangeof functionality. C# is a language to use with .NET. MSSQL - SQL Server is a databaseserver by Microsoft. The Microsoft relational database management systemis a softwareproduct which primarily stores and retrieves data requested by other applications. Internet Information Services (IIS) is one of the most powerful web servers fromMicrosoft that is used to host ASP.NET Web application
